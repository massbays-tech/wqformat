
# wqformat

<!-- badges: start -->
<!-- badges: end -->

The goal of wqformat is to provide simple R functions to convert common
water quality formats to that required for
[MassWateR](https://massbays-tech.github.io/MassWateR).

## Install and Load

You can install and load the development version of wqformat like so:

``` r
# install
install.packages('remotes')
remotes::install_github('massbays-tech/wq-format')

# load
library(wqformat)
```
